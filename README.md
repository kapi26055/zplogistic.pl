# ZPlogistic — GitHub Pages (wersja 1:1 na bazie screenów)

## Pliki
- `index.html` — Home Page
- `kontakt.html` — Kontakt
- `info-zarzad.html` — Info Zarząd
- `assets/css/style.css` — cały styl
- `assets/js/main.js` — mobilne menu
- `assets/img/` — grafiki (podmień na swoje)

## Podmiana zdjęć
Wgraj swoje pliki do `assets/img/` i nazwij dokładnie:
- `hero.jpg`
- `gallery-1.jpg`
- `gallery-2.jpg`
- `gallery-3.jpg`
- `footer.jpg`

## Formularz
W `kontakt.html` podmień:
`https://formspree.io/f/yourFormId`
na własny link z Formspree (darmowe).

## GitHub Pages
Repo → Settings → Pages → Deploy from branch → main / root.
Domena: Settings → Pages → Custom domain (np. www.zplogistic.pl).

1) Załóż darmowe konto na Formspree i utwórz formularz.
2) Skopiuj endpoint w stylu: `https://formspree.io/f/abcdwxyz`
3) Otwórz `kontakt.html` i podmień wartość `FORM_ENDPOINT`.
4) Zrób commit na GitHub — od razu działa.
